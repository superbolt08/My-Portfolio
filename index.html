My story

My name is Syed Saad Ali. A UBC computer science student dedicated towards taking my degree towards sustainable development. I initially pursued computer science with a passion for game development. However, I soon discovered the vast opportunities for interdisciplinary collaboration that technology offers. With this realization, I decided to channel my skills towards sustainable development, aiming to make a meaningful impact through innovation.

Why Sustainability?

A key goal in my career is to make a meaningful impact by contributing to sustainable development. Through my research, I discovered the vast potential of computer science in this field, from developing energy-efficient algorithms and using AI for natural disaster prediction to applying IoT and data science for environmental monitoring. These opportunities are incredibly exciting, and I am eager to explore them all. Sustainability is not just a passion; it's essential for a better future. Currently, I lead sustainability initiatives in my university's sustainability club, where I apply my skills and knowledge to drive positive change.

Experience

Dramania

Dramania is a web-based party game that showcases my skills in front-end development and interactive design. The game is built using HTML and CSS frameworks to ensure a responsive and visually appealing interface, while JavaScript handles the game logic, dynamically generating unique prompts for players.

Simon

Simon is a web-based memory game that challenges players to memorize and replicate a sequence of colors to progress through levels. The game incorporates interactive sounds and animations to create an engaging user experience, utilizing CSS for styling and jQuery for dynamic behavior and event handling. I developed this project while taking a web development course, which gave me valuable insights into integrating audio and visual components with user interactions. This experience enhanced my skills in front-end development and interactive design.
